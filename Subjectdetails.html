Subject Details

Subject Name: Principles of Programming Using C

Subject Code: BPOPS103/203

Description: This foundational course introduces students to the basics of computer programming using the C language, covering fundamental concepts such as data types, control structures, functions, arrays, pointers, and file handling.

Course Objectives

  • Understand the basic architecture and functionalities of a computer.
  • Apply programming constructs of the C language to solve real-world problems.

Course Outcomes

  • Describe the basic architecture and functionalities of a computer and identify its hardware components.
  • Utilize C programming constructs to address real-world problems.
  • Implement solutions using user-defined data structures like arrays for tasks such as searching and sorting.
  • Employ user-defined data structures like structures, unions, and pointers in problem-solving.
  • Design and develop solutions using modular programming constructs, including functions.

Course Structure

Module 1: Introduction to C

Overview of computers, input and output devices, and designing efficient programs. Introduction to C, structure of a C program, files used in a C program, compilers, compiling and executing C programs, variables, constants, and input/output statements in C.

Module 2: Operators in C

Operators in C, type conversion, and typecasting. Decision control and looping statements: Introduction to decision control, conditional branching statements, iterative statements, nested loops, break and continue statements, and the goto statement.

Module 3: Functions

Introduction to functions, function definition, declaration, calls, return statements, parameter passing, scope of variables, storage classes, and recursive functions. Arrays: Declaration, accessing elements, storing values, operations, passing arrays to functions, two-dimensional arrays, operations on two-dimensional arrays, passing two-dimensional arrays to functions, multidimensional arrays, and applications of arrays.

Module 4: Strings and Pointers

Introduction to strings, string taxonomy, operations on strings, miscellaneous string and character functions, and arrays of strings. Pointers: Introduction, declaring pointer variables, types of pointers, and passing arguments to functions using pointers.

Module 5: Structures, Unions, and Enumerated Data Types

Introduction to structures and functions, unions, unions inside structures, and enumerated data types. Files: Introduction to files, using files in C, reading and writing data files, and detecting the end of a file.
