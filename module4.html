Module 4: Important Questions in C Programming

1. What is a Pointer? Explain with an Example.

#include int main() { int a = 10; int *ptr = &a; printf("Value of a: %d\n", *ptr); return 0; }

2. Differentiate between Pointer and Array.

Pointers store memory addresses, while arrays store multiple values.

3. Dynamic Memory Allocation: malloc() and free()

#include #include int main() { int *ptr = (int *)malloc(5 * sizeof(int)); if (ptr == NULL) return 1; ptr[0] = 10; printf("First element: %d\n", ptr[0]); free(ptr); return 0; }

4. String Handling: gets() and puts()

#include int main() { char str[50]; gets(str); puts(str); return 0; }

5. String Manipulation Functions

#include #include int main() { char str[] = "Hello"; printf("Length: %d\n", strlen(str)); return 0; }

6. Reverse a String using Pointers

void reverseString(char *str);

7. Concatenate Two Strings

strcat(str1, str2);

8. Explain Structure in C with an Example

struct Student { char name[50]; int age; float marks; };

9. Differentiate between Structure and Union

Structures allocate memory for all members, while unions allocate memory for the largest member.

10. Copy One Structure to Another

struct Student s2 = s1;

11. What are wild pointers in C? How can they be avoided?

Wild pointers are uninitialized pointers that can cause unpredictable behavior. They can be avoided by initializing pointers to NULL.

12. Explain the concept of pointer arithmetic with an example.

#include int main() { int arr[] = {10, 20, 30}; int *ptr = arr; ptr++; printf("Second element: %d", *ptr); return 0; }

13. What is the use of calloc() in dynamic memory allocation?

calloc() is used for dynamic memory allocation and initializes allocated memory to zero.

14. Write a C program to count the number of vowels in a given string using pointers.

#include int countVowels(char *str) { int count = 0; while (*str) { if (*str == 'a' || *str == 'e' || *str == 'i' || *str == 'o' || *str == 'u') count++; str++; } return count; } int main() { char str[] = "hello"; printf("Vowel count: %d", countVowels(str)); return 0; }

15. How is a pointer to a structure used in C? Provide an example.

struct Student { char name[50]; int age; }; int main() { struct Student s = {"John", 20}; struct Student *ptr = &s; printf("%s %d", ptr->name, ptr->age); return 0; }

16. Explain the difference between memcpy() and strcpy() functions.

memcpy() copies a fixed number of bytes, whereas strcpy() copies a null-terminated string.

17. What is a function pointer? Write a program to demonstrate its usage.

#include void hello() { printf("Hello, World!"); } int main() { void (*ptr)() = hello; ptr(); return 0; }

18. How can an array of structures be used in C? Provide an example.

struct Student students[2];

19. Write a C program to compare two strings without using strcmp().

int compareStrings(char *str1, char *str2) { while (*str1 && (*str1 == *str2)) { str1++; str2++; } return *str1 - *str2; }
