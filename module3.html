Module 3: Important Questions in C Programming

1. What is an Array? Mention different types of arrays. How is a 2D array represented in memory?

An array is a collection of elements of the same data type stored in contiguous memory locations.

Types of arrays include:

  • One-Dimensional Array: A linear list of elements.
  • Two-Dimensional Array: A table of elements, often visualized as rows and columns.
  • Multi-Dimensional Array: Arrays with more than two dimensions.

A 2D array is stored in row-major order in C. Example:

int matrix[2][3] = { {1, 2, 3}, {4, 5, 6} };

2. Differentiate between Call by Value and Call by Reference

Call by Value:

void increment(int a) { a = a + 1; } int main() { int x = 5; increment(x); // x remains 5 }

Call by Reference:

void increment(int *a) { *a = *a + 1; } int main() { int x = 5; increment(&x); // x becomes 6 }

3. Representation of One-Dimensional Integer Array

A one-dimensional integer array is stored in contiguous memory locations. Example:

int arr[5] = {1, 2, 3, 4, 5};

6. C Program to Find Factorial Using Recursion

#include int factorial(int n) { if (n == 0) return 1; else return n * factorial(n - 1); } int main() { int num = 5; printf("Factorial of %d is %d\n", num, factorial(num)); return 0; }

7. C Program to Find GCD and LCM

#include int gcd(int a, int b) { if (b == 0) return a; return gcd(b, a % b); } int lcm(int a, int b) { return (a * b) / gcd(a, b); } int main() { int num1 = 12, num2 = 18; printf("GCD of %d and %d is %d\n", num1, num2, gcd(num1, num2)); printf("LCM of %d and %d is %d\n", num1, num2, lcm(num1, num2)); return 0; }

8. Explain Recursion and Fibonacci Series Program

Recursion is a programming technique where a function calls itself to solve smaller instances of the same problem.

#include int fibonacci(int n) { if (n <= 1) return n; return fibonacci(n - 1) + fibonacci(n - 2); } int main() { int n = 10; for (int i = 0; i < n; i++) { printf("%d ", fibonacci(i)); } return 0; }
