Module 5 Questions

1. What is File Handling in C?

File handling in C enables storing data in a file instead of standard input/output. It provides operations like creating, reading, writing, and closing files.

2. What are the different modes in file handling?

Modes include: r (read), w (write), a (append), r+ (read & write), w+ (write & read), a+ (append & read).

3. How to open a file in C?

FILE *fp; fp = fopen("file.txt", "r");

4. Write a program to read a file in C.

#include int main() { FILE *fp; char ch; fp = fopen("test.txt", "r"); if (fp == NULL) { printf("File not found!\n"); return 1; } while ((ch = fgetc(fp)) != EOF) putchar(ch); fclose(fp); return 0; }

5. What is the difference between fgetc() and fgets()?

fgetc() reads a single character, while fgets() reads a string from a file.

6. How to write to a file in C?

FILE *fp; fp = fopen("file.txt", "w"); fprintf(fp, "Hello, World!"); fclose(fp);

7. Write a program to copy contents from one file to another.

#include int main() { FILE *src, *dest; char ch; src = fopen("source.txt", "r"); dest = fopen("destination.txt", "w"); while ((ch = fgetc(src)) != EOF) fputc(ch, dest); fclose(src); fclose(dest); return 0; }

8. What is the use of rewind() in file handling?

rewind() sets the file position to the beginning of the file.

9. Explain the difference between text files and binary files.

Text files store human-readable characters, while binary files store data in machine-readable format.

10. What is ftell() and fseek()?

ftell() returns the current file position, and fseek() moves the file pointer to a specific location.

11. Write a program to append data to a file.

FILE *fp; fp = fopen("file.txt", "a"); fprintf(fp, "Appended Text\n"); fclose(fp);

12. How to check if a file exists before opening it?

FILE *fp = fopen("file.txt", "r"); if (fp) { printf("File exists\n"); fclose(fp); } else { printf("File does not exist\n"); }

13. How do you delete a file in C?

remove("file.txt");

14. What is the difference between fread() and fwrite()?

fread() reads binary data, and fwrite() writes binary data.

15. Write a program to store student data in a file.

#include struct Student { char name[50]; int age; }; int main() { FILE *fp = fopen("students.dat", "wb"); struct Student s = {"Alice", 20}; fwrite(&s, sizeof(s), 1, fp); fclose(fp); return 0; }

16. How to read a structure from a binary file?

FILE *fp = fopen("students.dat", "rb"); struct Student s; fread(&s, sizeof(s), 1, fp); fclose(fp);

17. Explain the difference between fprintf() and fwrite().

fprintf() writes formatted text, while fwrite() writes raw binary data.

18. How can you lock a file in C?

Use flock() or fcntl() to apply a lock.

19. What is a command-line argument? Write a program using it.

#include int main(int argc, char *argv[]) { printf("Argument: %s\n", argv[1]); return 0; }

20. What is stderr, stdin, and stdout?

stderr is for error output, stdin for input, and stdout for output.
