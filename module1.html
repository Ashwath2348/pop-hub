Computer Fundamentals

Components of a Computer

A computer consists of five main components:

  • Input Unit – Takes input from the user (e.g., keyboard, mouse).
  • Central Processing Unit (CPU) – Processes data and consists of:
    • Control Unit (CU) – Directs the operations of the computer.
    • Arithmetic Logic Unit (ALU) – Performs mathematical and logical operations.
  • Memory Unit – Stores data temporarily or permanently.
  • Output Unit – Displays the results (e.g., monitor, printer).
  • Storage Unit – Stores data permanently (e.g., HDD, SSD).

Block Diagram:

Input Unit → CPU (CU + ALU) → Memory Unit → Output Unit

Evolution of Computers

  • First Generation (1940-1956): Used vacuum tubes, slow, large, expensive (e.g., ENIAC).
  • Second Generation (1956-1963): Used transistors, smaller, faster, more efficient.
  • Third Generation (1964-1971): Used Integrated Circuits (ICs), improved speed and reliability.
  • Fourth Generation (1971-Present): Used microprocessors, personal computers emerged.
  • Fifth Generation (Present & Beyond): Focus on AI, quantum computing, parallel processing.

Basic Terminology

  • Bit – The smallest unit of data (0 or 1).
  • Byte – 8 bits, used to represent characters.
  • Word – A fixed number of bytes a CPU processes at a time (e.g., 32-bit, 64-bit).
  • Kilobyte (KB) – 1 KB = 1024 bytes.
  • Megabyte (MB) – 1 MB = 1024 KB.

Types of Memory

  • Primary Memory: Fast, volatile, directly accessed by CPU.
    • RAM – Stores data temporarily.
    • ROM – Stores firmware, non-volatile.
  • Secondary Memory: Permanent storage (e.g., HDD, SSD).
  • Cache Memory: Small, high-speed memory for frequently used data.
  • Virtual Memory: Uses HDD/SSD space as extra RAM when needed.

Classification of Computers

  • Supercomputers: Extremely fast, used for scientific research (e.g., PARAM).
  • Mainframe Computers: Large, multi-user, used in banking and airlines.
  • Minicomputers: Mid-sized, used in small industries.
  • Microcomputers (PCs): Personal use, desktops, laptops, tablets.

Input and Output Devices

Input Devices

  • Keyboard
  • Mouse
  • Scanner
  • Joystick
  • Microphone

Output Devices

  • Monitor (LCD, LED)
  • Printer (Inkjet, Laser)
  • Speaker
  • Projector

Computer Networks

  • Network: Connection of computers for data exchange.
  • LAN (Local Area Network): Small area like offices, homes.
  • WAN (Wide Area Network): Large-scale networks (e.g., Internet).
  • MAN (Metropolitan Area Network): City-wide networks.
  • Network Topology: Arrangement of network components (Star, Ring, Bus).
