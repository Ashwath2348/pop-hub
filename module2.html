Module 2: Important Questions in C Programming

1. Explain If-Else Statement

int num = 10; if (num > 0) { printf("Number is positive\n"); } else { printf("Number is not positive\n"); }

2. Explain While Loop with an Example

int i = 1; while (i <= 5) { printf("%d ", i); i++; }

3. Explain Do-While Loop with an Example

int i = 1; do { printf("%d ", i); i++; } while (i <= 5);

4. Explain For Loop with an Example

for (int i = 1; i <= 5; i++) { printf("%d ", i); }

5. What is a Switch-Case Statement? Explain with an Example

int choice = 2; switch (choice) { case 1: printf("Choice is 1\n"); break; case 2: printf("Choice is 2\n"); break; default: printf("Invalid choice\n"); }

6. What are Break and Continue Statements? Explain with Examples

for (int i = 1; i <= 5; i++) { if (i == 3) continue; printf("%d ", i); }

7. Explain Type Casting in C with an Example

float num = 10.6; int newNum = (int) num; printf("Casted value: %d\n", newNum);

8. Write a C Program to Calculate the Area and Perimeter of a Rectangle

int length = 10, breadth = 5; int area = length * breadth; int perimeter = 2 * (length + breadth); printf("Area: %d, Perimeter: %d\n", area, perimeter);

9. Define an Algorithm and Write an Algorithm to Find the Factorial of a Number

int factorial(int n) { if (n == 0) return 1; return n * factorial(n - 1); }

10. Explain the Different Types of Operators in C

  • Arithmetic Operators: +, -, *, /, %
  • Relational Operators: ==, !=, >, <, >=, <=
  • Logical Operators: &&, ||, !
  • Bitwise Operators: &, |, ^, ~, <<, >>
  • Assignment Operators: =, +=, -=, *=, /=

11. Write a C Program to Check if a Number is Even or Odd

int num = 10; if (num % 2 == 0) printf("Even\n"); else printf("Odd\n");

12. What are the Different Data Types in C? Explain with Examples

  • int - Integer values (e.g., int a = 10;)
  • float - Decimal values (e.g., float pi = 3.14;)
  • char - Single character (e.g., char ch = 'A';)
  • double - Double-precision floating-point numbers
